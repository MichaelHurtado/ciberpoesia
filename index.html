El libro como universo - el universo como libro

  • Un libro no es el lugar donde sólo se almacenan palabras.
  • Un libro es una secuencia de espacios, momentos y vivencias.
  • Un libro incorpora espacio, tiempo y signo.
  • Un libro no se crea ni se destruye.
  • Un libro incluye a otros libros.
  • Un libro puede adoptar las formas más inesperadas.
  • El conjunto de todos los libros es un libro.
  • El conjunto de todo lo que no es un libro es un libro.
  • El universo entero es un libro.
  • El libro puede tomar la forma de un tweet, una plataforma, una aplicación, un sitio web o lo que sea.
